Panduan Penggunaan

Untuk Karyawan

  1. Klik "Employee Attendance"
  2. Login dengan akun yang diberikan HR
  3. Izinkan akses kamera & lokasi
  4. Ambil foto selfie untuk verifikasi
  5. Submit absensi masuk/pulang

Untuk Admin/HR

  1. Klik "Admin Dashboard"
  2. Login dengan akun admin
  3. Kelola karyawan via "Kelola Karyawan"
  4. Monitor absensi real-time
  5. Lihat foto & lokasi karyawan
  6. Export data ke Excel

Install Aplikasi (PWA)

  • • Desktop: Klik icon install di address bar
  • • Android: Menu → Add to Home Screen
  • • iOS: Share → Add to Home Screen
